01

Synthetic Data Generation

2,000 training samples are generated using established geotechnical empirical correlations from Terzaghi, Meyerhof, Hazen, Wolff, and others. Gaussian noise is added for realism.

02

Neural Network Training

A TensorFlow.js neural network with 2 hidden layers (64 and 32 neurons, ReLU activation) is trained directly in your browser for ~100 epochs.

03

Instant Prediction

Your input parameters are normalized and fed through the trained model. The network captures non-linear interactions between soil properties for nuanced predictions.

Disclaimer: This tool uses a neural network trained on synthetic data from empirical correlations. It is intended for educational and preliminary assessment purposes only. Results should not replace proper site investigation and laboratory testing for final engineering design.
